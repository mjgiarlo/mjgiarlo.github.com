Ostensive Relevance, cont.

  • In the OM, relevance feedback is provided implicitly “by interpreting a user’s selection of one document over others as an indication that this document is more relevant.”

  • The query submitted to the system is learned and continues to be adapted based on the features of the images to which the user browses

  • Ostensive Relevance is thus defined: “The Ostensive Relevance of an information object is the degree to which evidence from the object is representative / indicative of the current information need
