FEDORA, continued

  • FEDORA requires Java and a database management system, such as MySQL, and runs on Windows, Linux, and UNIX platforms.

  • FEDORA runs as a web service, with separate interfaces for access and management to ensure security for users and content providers.

  • Metadata harvesting (OAI-PMH) and repository-to-repository (R2R) capabilities are built in to present an open, interoperable archive.
