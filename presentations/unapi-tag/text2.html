unAPI: Just the facts, ma'am.

  • An open specification conformed to ROGUE '05 standard

  • A lightweight, tiny even, HTTP API providing a service for clipboard-like copying and pasting of digital information objects published on the web

  • But don't we already have this in...

    • OpenURL?
    • OAI-PMH?
    • SRU? / OpenSearch?
    • Atom Syndication Format and Publishing Protocol?

  • In fact, unAPI designed with layering on these pre-existing protocols
