Systems Evaluated

  • Pure Ostensive Browser (POB) – An exclusively “point-and-click” image browser, in which the system performs all query adaptation and feature weighting

  • Controlled Ostensive Browser (COB) – Like the POB, but with options for feature selection and weighting, and also for changing the terms selected by the system

  • Manual Query System (MQS) – A more traditional image retrieval system providing a search box and an image selection area
