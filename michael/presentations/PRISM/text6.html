Metadata Harvesting (OAI-PMH)

  • Metadata harvesting consists of both exposure and aggregation of metadata; that is, in order for it to be harvested, it must first be made available. Native schema is “OAI Dublin Core.”

  • In a metadata harvesting framework such as the OAI-PMH, there are two classes of resources:

    • Data Providers, repositories that expose their collections' metadata
    • Service Providers, services that harvest metadata from data providers, enabling users to search across discrete collections
