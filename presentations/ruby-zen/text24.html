  • In parallel, there have been efforts to rewrite the Ruby language in Java (JRuby) and .NET (IronRuby) – the original is written in C

    • Why bother? JRuby allows you to run Ruby code on the JVM, which is a common development environment in the enterprise, so it expands Ruby's reach considerably.
    • Perhaps more interesting, it allows you to „reach into“ Java in your Ruby code, giving full access to Java libraries in Ruby.
